As soon as the script/btgcli is done loading the torrent file, it can
be deleted, as the daemon loaded it and will take care of downloading.

Please try to read and understand:
http://btg.berlios.de/howto.html#supported-directory-structure

> Can i check torrent download from btgcli with btgdaemon and its web

> 
> Or i need to use ssh to watch progress from another computer?

After the script loads a torrent, you can check the status with the www
interface or use one of the other clients. Just make sure that you use
the same session. You can use all the clients at the same time to
access the same session.

Please try to read the documentation, it does explain how BTG works. If
not, add a bug and I will try to make the docs more clear. :)
