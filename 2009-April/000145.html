I've attempted to build with external xmlrpc-epi and I
found I needed a small patch since the headers are now in
/usr/include/xmlrpc-epi:
diff -up btg-1.0.0/bcore/externalization/xmlrpc.h.orig btg-1.0.0/bcore/externalization/xmlrpc.h
--- btg-1.0.0/bcore/externalization/xmlrpc.h.orig       2009-04-21 16:31:55.000000000 +0200
+++ btg-1.0.0/bcore/externalization/xmlrpc.h    2009-04-21 16:32:07.000000000 +0200
@@ -32,7 +32,7 @@ extern "C"
 #if BTG_INTERNAL_XMLRPC
 #  include <external/xmlrpc-epi/src/xmlrpc.h>
 #else
-#  include <xmlrpc.h>
+#  include <xmlrpc-epi/xmlrpc.h>
 #endif
 }

The configure test should probably be updated aswell otherwise one needs to
set XMLRPC_CFLAGS="-I/usr/include/xmlrpc-epi" to have it detect that external
xmlrpc-epi is available.
