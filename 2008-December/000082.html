Hi,

I have a Buffalo Linkstation Live with an ARM processor and I am currently 
running BTG (daemon only + web UI).
I have installed Debian on the LS, and upgraded it to the "testing" branch.
I used libtorrent and libboost packages provided by standard apt repositories.
Then I was able to _natively_ compile BTG on the LS; I compiled just 
btgdaemon, btgcli and the web UI using the following options:

./configure --enable-btg-config --disable-gui --disable-viewer --enable-cli --
enable-www --disable-upnp --enable-session-saving --enable-event-callback --
enable-url

It took 4-5 hours. (BTG 0.9.9rc1)

Unfortunately I was not able to get cross-compilation up and running.

Hope it helps.
