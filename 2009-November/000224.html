The following python script seems to work.

"
#!/usr/bin/python

import xmlrpclib

uri = "http://localhost:16000"
p = xmlrpclib.ServerProxy(uri)
p.general.initconnection(1, "user","hash of password")
"

The daemon prints:

[2009-Nov-06 23:29:36] hdl: client (0): (g) Init connection.
[2009-Nov-06 23:29:36] hdl: daemon (0): (g) Ack.
